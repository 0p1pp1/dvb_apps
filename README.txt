配布の構成について
 ./cmds/                各種ユーティリティコマンド 主にストリーム解析用
 ./scripts/             各種ユーティリテイ スクリプト 主に録画用
 ./readme-cmds.txt       ./cmds/* の説明
 ./readme-scripts.txt     ./scripts/* の説明
 ./README.txt これ

 詳しくはreadm-*.txtと、各コマンドの-hオプション、ソースコードを参照
